### Prerequisitos

* NPM
```sh
$ sudo apt-get install npm
```
* johnny-five
```sh
$ sudo npm install johnny-five
```
* http
```sh
$ sudo npm install http
```
* socket.io
```sh
$ sudo npm install socket.io
```

### Ejemplo de uso
```sh
$ git clone https://github.com/andyalonzo/arduino-web-panel.git
$ cd LED
$ node led.js

```
